### `index.html` — THE HERO
**[→ phenomenal-mark-site.vercel.app](https://phenomenal-mark-site.vercel.app)**

The landing. A full-screen living multiverse built in vanilla Three.js + Canvas 2D.

| Feature | Description |
|---|---|
| **Living Multiverse Tree** | 1,000+ boid particles forming a cosmic tree — trunk, fibonacci branches, galaxy clusters at tips. Cursor interaction: attract on hover, explode on click. |
| **Holographic PHENOMENAL Text** | Canvas particle system that samples letter shapes pixel-by-pixel. Spring physics hold letters together; hover scatters them. Holographic color cycles per particle. |
| **Scroll-Reactive Particles** | As you scroll, particles stretch and motion-blur downward, then regroup into the tree formation on the next section. |
| **MARK Glitch Text** | JetBrains Mono with randomized ASCII corruption — cyan/violet split-channel effect. |
| **20-Year Timeline** | Complete career arc from 2004–2026, scroll-revealed with IntersectionObserver. Every entry links to the verified work. |

| **Disciplines Grid** | Four portal cards — Fashion, Art, Cinema, Tech — each navigates to its dedicated page. |

### `world.html` — THE 3D WORLD
**[→ /world.html](https://phenomenal-mark-site.vercel.app/world.html)**

A navigable Three.js environment. Enter through the hero.

| Feature | Description |
|---|---|
| **Neon Hooded Figure** | Built from Three.js primitives (CylinderGeometry, ConeGeometry, SphereGeometry) with EdgesGeometry TRON-style circuit glow. Follows your cursor across the X axis in real time. |
| **Walk Animation** | Figure walks with oscillating leg/arm cycle. Breathing idle animation. |
| **Footstep Sounds** | Web Audio API procedural noise — bandpass-filtered, gain-enveloped footstep synthesis. Triggers every 0.38s while moving. |
| **Day / Night Mode** | Scene lighting automatically matches your local timezone: hour 6–20 = day sky; else void-black night with star field. |
| **6 Monolith Portals** | Tall geometric slabs on the ground arc — hover to shimmer, click to trigger figure attack animation. Figure randomly kicks, punches, or fires an energy beam to shatter the monolith, then transitions to the target page. |
| **Monolith Destinations** | `HERO` · `ART` · `CINEMA` · `FASHION` · `FORECAST` · `THE SWARM (Living Swarm Demo)` |

## III. TECHNICAL STACK

```
Hosting:    Vercel (production) + GitHub Pages (mirror)
Rendering:  Vanilla HTML · CSS · JavaScript — zero build step
3D Engine:  Three.js r128 (CDN)
Physics:    Custom boid simulation (flat Float32Array for performance)
Audio:      Web Audio API — procedural footstep synthesis
Shaders:    Custom GLSL vertex + fragment (star field, nebula)
Payments:   Stripe Payment Links (no server required)
Fonts:      Outfit · Cormorant Garamond · JetBrains Mono (Google Fonts)
Auth:       Ethereum wallet signature (ethers.js v5) for admin panel
```

## IV. PROTECTION

All pages enforce:
- No right-click / context menu
- No F12 / DevTools keyboard shortcut
